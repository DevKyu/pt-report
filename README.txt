물리치료 집계 자동화
====================

처방 데이터를 자동으로 집계하여 Excel 리포트와 HTML 대시보드를 생성합니다.
별도의 Python 설치 없이 바로 사용할 수 있습니다.


[ 첫 실행 ]

  1. ptreport.exe 더블클릭
  2. 바탕화면에 "PT Report" 바로가기가 자동 생성됩니다.
  3. 자동 실행 스케줄이 등록됩니다 (약 1초 후 로그에 표시)

  이후로는 바탕화면 "PT Report" 아이콘으로 앱을 여세요.


[ 파일 넣기 ]

  data\weekly\   폴더  ->  주별 처방 데이터  (매주 월요일 전)
  data\monthly\  폴더  ->  월별 전체 데이터  (매월 1일 전)

  * 파일 형식: .xlsx / .xls 모두 지원 (대소문자 무관, .XLS 도 인식)
  * 여러 주차 파일을 한번에 넣어도 됩니다.


[ 주별 집계 동작 방식 ]

  주차별 파일을 넣고 실행할 때마다 이전 주차 데이터가 누적됩니다.

  예시:
    1주차 파일 넣고 실행  →  1주차 집계 리포트
    2주차 파일 넣고 실행  →  1+2주차 누적 집계 리포트
    3주차 파일 넣고 실행  →  1+2+3주차 누적 집계 리포트

  처리 완료된 파일은 data\weekly\_done\ 폴더로 자동 이동됩니다.
  이 폴더를 지우지 마세요 — 누적 집계에 사용됩니다.


[ 자동 실행 스케줄 ]

  주별 : 매주 월요일 12:00  (PC가 켜져 있으면 자동 처리)
  월별 : 매월  1일   12:00  (PC가 켜져 있으면 자동 처리)

  * 스케줄 시간에 PC가 꺼져 있었다면 앱에서 수동 실행하세요.


[ 결과 확인 ]

  output\YYYY-MM\reports\    ->  Excel 리포트
  output\YYYY-MM\dashboard\  ->  HTML 대시보드 (브라우저로 열기)

  앱 화면의 [Excel] [대시보드] 버튼으로 바로 열 수 있습니다.
  HTML 대시보드 차트는 인터넷 연결 시 정상 표시됩니다.
  (오프라인 환경에서는 표·텍스트는 보이지만 차트는 표시되지 않음)


[ 수동 실행 ]

  앱 화면에서 [주별 실행] 또는 [월별 실행] 버튼 클릭


[ 월별 정합성 검증 ]

  월별 실행 시 data\monthly\ 의 월별 파일과
  누적된 주별 파일 합산을 자동으로 비교합니다.

  결과가 Excel 리포트의 [정합성 검증] 시트에 표시됩니다.
  불일치가 있으면 원본 데이터를 확인하세요.


[ 주의사항 ]

  - 처리 완료된 원본 파일은 data\_done\ 폴더로 자동 이동됩니다.
  - 실행 기록은 run.log 파일에서 확인할 수 있습니다.
  - 같은 파일을 다시 넣어도 중복 집계되지 않습니다 (행 단위 중복 제거).


[ 제거 방법 ]

  uninstall.bat  우클릭 -> 관리자 권한으로 실행
  -> 자동 실행 스케줄 삭제됩니다.
  이후 이 폴더 전체를 삭제하세요.
